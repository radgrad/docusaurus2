Setup the IDE

For RadGrad development, we recommend IntelliJ IDEA. IntelliJ IDEA has excellent support for Meteor application development.

To configure IntelliJ for RadGrad development, you may want to follow the ICS 314 instructions.

The ics-se-code-style file configures IntelliJ with the coding style checked by ESLint.
